#  MentorMatch

Платформа менторства со свайп-механикой как в Tinder. Студенты и выпускники
находят друг друга свайпами, при взаимном лайке получают **мэтч** и могут
общаться в чате — помощь в карьере и учёбе без «холодных» писем.

- **Backend:** Python + Flask + SQLite (без ORM, только stdlib `sqlite3`)
- **Frontend:** ванильные HTML / CSS / JS (без сборки)
- **Дизайн:** тёплая приглушённая палитра — плюм `#4B4453`, лаванда `#B5A5C7`,
  розово-коричневый `#9E7970`, тёмно-коричневый `#6A4840`, серый `#838185`.
  Адаптив 375 → 1440px, мягкие пружинные анимации.
- **Типографика:** вордмарк «**Mentor**» — Climate Crisis, «*Match*» — WindSong;
  основной текст — Cosola Mono (если установлен локально) с фолбэком на
  JetBrains Mono (поддержка кириллицы). Все три — через Google Fonts.
- **3D-планета:** на экране входа/регистрации вращается интерактивный глобус
  ([cobe](https://github.com/shuding/cobe), грузится с CDN), показывающий маркеры
  городов и связь «ментор ↔ выпускник». Перетаскивается мышью/пальцем; при
  `prefers-reduced-motion` и без сети — корректный фолбэк.

## Возможности

- Регистрация студента или ментора с профилем (сфера, навыки, цель, био)
- Свайп-дека: drag-жесты мышью/пальцем, штампы «Интересно/Пропуск», кнопки like/skip/инфо
- Подбор противоположной роли, уже просмотренные профили не повторяются
- Мэтч при взаимном лайке + модалка «Это мэтч!»
- Список мэтчей и чат 1-на-1
- Редактирование своего профиля

## Запуск

**В один клик:** запусти `start.bat` (Windows) или `start.sh` (Mac/Linux) из папки проекта —
он поставит зависимости, поднимет сервер и откроет браузер.

Вручную:

```bash
cd backend
python -m pip install -r requirements.txt
python app.py
```

Открой **http://localhost:5000**

> ⚠️ Сайт нужно открывать **через сервер** по адресу `http://localhost:5000`.
> Если открыть `index.html` двойным кликом (`file://`), вход и регистрация выдадут
> ошибку **«Failed to fetch»** — браузер не видит backend.

База `mentormatch.db` и демо-данные создаются автоматически при первом запуске.

## Структура

```
backend/
  app.py          — Flask: REST API (/api/*) + раздача фронтенда
  db.py           — подключение к SQLite, инициализация
  schema.sql      — схема БД (users, swipes, matches, messages)
  seed.py         — демо-пользователи
  requirements.txt
frontend/
  index.html      — каркас SPA
  css/styles.css  — дизайн-токены и стили
  js/app.js       — вся логика клиента (роутинг, свайпы, чат)
```

## API (вкратце)

| Метод | Путь | Назначение |
|-------|------|-----------|
| POST | `/api/register` | регистрация |
| POST | `/api/login` · `/api/logout` | сессия |
| GET/PUT | `/api/me` | свой профиль |
| GET | `/api/deck` | кандидаты на свайп |
| POST | `/api/swipe` | `{target_id, direction}` → `{match}` |
| GET | `/api/matches` | список мэтчей |
| GET/POST | `/api/messages/<match_id>` | переписка |

## Заметки по продакшену

Это MVP для демонстрации. Перед боевым использованием стоит: сменить
`MENTORMATCH_SECRET`, перейти на полноценную БД (PostgreSQL), добавить
HTTPS/CSRF-защиту, пагинацию деки и реальный realtime для чата (WebSocket).
